EverTrust Homecare Services - Deployable React Site

This ZIP matches the React/Tailwind design we created in ChatGPT canvas.

How to deploy on Vercel:
1. Unzip the folder
2. Upload the folder to a new GitHub repository, or import it directly into Vercel
3. Framework preset: Vite
4. Build command: npm run build
5. Output directory: dist

How to deploy on Netlify:
1. Unzip the folder
2. Upload the folder to GitHub or drag the project into Netlify
3. Build command: npm run build
4. Publish directory: dist

How to deploy on GitHub Pages:
1. Push the project to GitHub
2. Build locally with npm install then npm run build
3. Publish the /dist folder using GitHub Pages

Local development:
1. npm install
2. npm run dev

Files to customize:
- src/App.jsx for page content
- Replace placeholder phone number and city
- Replace stock images with your branded images
